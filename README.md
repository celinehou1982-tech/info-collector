# 信息收集与整理工具

一个智能的信息收集和自动整理工具，支持Mac和iPhone使用。

## 项目概述

这是一个基于Web技术开发的PWA应用，帮助您高效地收集、分类和管理来自不同来源的信息内容。

### 当前版本功能 (v0.1 - MVP)

- ✅ 目录管理：创建多级目录，为每个目录设置关键词
- ✅ 内容添加：支持文本输入和URL输入
- ✅ 自动分类：基于关键词自动匹配目录
- ✅ 内容管理：浏览、查看、编辑和删除内容
- ✅ 本地存储：使用IndexedDB在浏览器本地存储数据
- ✅ 标签系统：为内容添加自定义标签
- ✅ Markdown支持：内容使用Markdown格式显示
- ✅ 响应式设计：支持桌面和移动端

### 开发中的功能

- 🔄 网页抓取：自动抓取网页内容（需要后端服务）
- 🔄 AI摘要：自动生成内容摘要和核心观点
- 🔄 汇总文档：生成分类汇总文档
- 🔄 数据导出：支持多种格式导出
- 🔄 搜索功能：全文搜索和高级筛选
- 🔄 PWA功能：安装到桌面，支持离线使用

## 技术栈

### 前端
- React 18 + TypeScript
- Vite 5
- Material-UI (MUI) v5
- Zustand (状态管理)
- Dexie.js (IndexedDB)
- react-markdown (Markdown渲染)

### 后端 (计划中)
- Node.js + Express
- TypeScript
- Cheerio (HTML解析)

## 快速开始

### 前置要求

- Node.js 18.x 或更高版本
- npm 或 yarn

### 安装和运行

1. **克隆项目**
   ```bash
   cd /Users/houjie5/info-collector
   ```

2. **安装依赖**
   ```bash
   cd frontend
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **访问应用**
   打开浏览器访问 http://localhost:5173/

### 构建生产版本

```bash
npm run build
```

构建产物会在 `dist` 目录下。

## 使用指南

### 1. 创建目录

1. 在左侧边栏点击 "+" 按钮
2. 输入目录名称和描述
3. 添加关键词（用于自动分类）
4. 点击保存

### 2. 添加内容

1. 点击右下角的 "+" 浮动按钮
2. 选择"文本"或"链接"标签
3. 输入标题和内容
4. 添加标签（可选）
5. 点击保存

系统会根据内容自动匹配合适的目录。

### 3. 浏览内容

- 点击左侧目录查看该目录下的内容
- 点击内容卡片查看详情
- 在详情页可以编辑或删除内容

### 4. 数据管理

目前数据存储在浏览器的IndexedDB中：
- 数据不会丢失（除非清除浏览器数据）
- 暂不支持跨设备同步（计划中）

## 故障排除

### 应用无法启动

确保Node.js版本正确：
```bash
node --version  # 应该是 v18.x 或更高
```

### 数据丢失

数据存储在浏览器IndexedDB中，注意：
- 不要清除浏览器数据
- 使用无痕模式时数据不会保存
- 建议定期导出备份（功能开发中）

### 界面显示异常

尝试清除浏览器缓存并刷新页面。
