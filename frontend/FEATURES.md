# 🎯 Catch4You - 智能信息收集与整理工具

**一站式解决您的信息管理需求**

访问地址：[https://catch4you.netlify.app](https://catch4you.netlify.app)

---

## ✨ 核心特性

### 📥 多样化信息输入

**支持5种输入方式，满足各种场景需求：**

- **📝 文本输入** - 直接粘贴文字内容，支持Markdown格式
- **🔗 链接抓取** - 自动抓取网页内容，智能提取正文
  - 支持微信公众号文章
  - 自动转换为Markdown格式
  - 智能提取标题、作者、发布日期
- **📸 图片识别** - OCR文字识别技术
  - 上传图片自动提取文字
  - 支持截图、照片等多种图片格式
- **📄 PDF解析** - 一键提取PDF文本
  - 快速提取PDF文档内容
  - 保留文本结构
- **📰 订阅管理** - 自动信息采集
  - 定期抓取关注公司的最新动态
  - 预设Tesla、OpenAI、Anthropic等热门公司
  - 支持RSS、HackerNews等多种来源

### 🗂️ 智能分类系统

**让信息井井有条：**

- **目录树管理** - 自由创建多级目录结构
- **关键词匹配** - 基于关键词自动分类
  - 配置关键词权重
  - 支持同义词识别
  - 智能推荐分类
- **标签系统** - 灵活的标签管理
  - 标签云可视化
  - 快速筛选和定位

### 🤖 AI增强功能

**让AI成为您的信息助手：**

- **智能摘要** - 一键生成内容摘要
  - 支持OpenAI、Claude等多种AI模型
  - 长文本自动分块处理
  - 用户自行配置API密钥
- **核心观点提取** - 快速把握文章要点
  - 自动提取3-5个关键观点
  - 支持手动编辑调整
- **汇总文档生成** - 整合知识库
  - 按目录、时间、标签筛选
  - 生成结构化汇总文档
  - 支持Markdown、HTML、PDF导出

### 🔍 强大的搜索功能

**快速找到您需要的信息：**

- **全文搜索** - 在所有内容中查找关键词
- **多维度筛选**
  - 按目录筛选
  - 按标签筛选
  - 按时间范围筛选
  - 组合条件搜索
- **标签云** - 可视化热门标签

### 📱 跨平台支持

**随时随地访问您的知识库：**

- **PWA应用** - 可安装到手机主屏幕
- **响应式设计** - 完美适配手机、平板、电脑
- **离线访问** - Service Worker支持离线浏览
- **本地存储** - 数据存储在浏览器IndexedDB中
- **数据导出/导入** - 支持备份和迁移

### 💎 实用功能

- **Markdown编辑器** - 所见即所得的编辑体验
- **实时预览** - 即时查看格式化效果
- **代码高亮** - 支持多种编程语言
- **图片预览** - 内置图片查看器
- **暗色模式** - 保护您的眼睛（可选）

## 🛠️ 技术栈

- **前端**: React 18 + TypeScript + Material-UI
- **存储**: IndexedDB (Dexie.js)
- **PWA**: Service Worker + Manifest
- **部署**: Netlify (Serverless Functions)
- **AI**: OpenAI API (可选)

## 📖 快速开始

1. **访问** https://catch4you.netlify.app
2. **点击** 右下角的"+"按钮
3. **选择** 输入方式（文本/链接/图片/PDF）
4. **输入** 内容并保存
5. **开始** 管理您的知识库！
